# Images Directory

This directory contains all image assets for the NFT Marketplace.

## Directory Structure

- `artworks/` - NFT artwork images
- `artists/` - Artist profile pictures
- `featured/` - Featured NFT images

## Image Sources

Currently using Unsplash placeholder images. Replace with your own images by:

1. Adding images to the respective directories
2. Updating component imports to use local paths: `/images/artworks/image-name.jpg`

## Image Optimization

- Recommended format: WebP or optimized JPG/PNG
- Recommended size: 
  - Artworks: 500x500px to 1000x1000px
  - Artist avatars: 200x200px to 400x400px
  - Featured images: 600x600px to 1200x1200px
